# Databuddy Skills

Agent skills for [Databuddy](https://databuddy.cc) — privacy-first analytics SDK.

## Install

```bash
npx skills add databuddy-analytics/skills
```

## Skills

| Skill | Description |
|-------|-------------|
| [databuddy](./databuddy/) | SDK overview — browser tracking, server events, feature flags, REST API |
| [databuddy-core](./databuddy/databuddy-core/) | Core browser tracking utilities and config |
| [databuddy-react](./databuddy/databuddy-react/) | React/Next.js component and hooks |
| [databuddy-vue](./databuddy/databuddy-vue/) | Vue 3 component, plugin, and composables |
| [databuddy-node](./databuddy/databuddy-node/) | Server-side tracking with batching and middleware |
| [databuddy-flags](./databuddy/databuddy-flags/) | Feature flags for React, Vue, and Node.js |
| [databuddy-events](./databuddy/databuddy-events/) | Event design — naming, properties, taxonomies |
| [databuddy-api](./databuddy/databuddy-api/) | REST API for querying analytics data |
