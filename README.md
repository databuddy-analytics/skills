# Databuddy Skills

Agent skill for [Databuddy](https://databuddy.cc) — privacy-first analytics SDK, feature flags, DevTools, and public APIs.

## Install

```bash
npx skills add databuddy-analytics/skills
```

## Skill

| Skill | Description |
|-------|-------------|
| [databuddy](./databuddy/) | External integration guidance for browser tracking, server events, feature flags, DevTools, REST analytics, event ingestion, attribution, and event design |
